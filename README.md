# S2T 2026 – Feedack Terminal

S2T 2026 –  Feedback Terminal is an interactive web application designed for student events and competitions. The platform allows participants to submit feedback, select event images from Google Drive, and generate a personalized poster in real time using HTML Canvas.

## Features
- Event-wise image selection from Google Drive  
- Live poster preview (1080×1350) with template overlay  
- Auto-fit text for school and student names  
- Rounded image placeholders and modern UI  
- One-click poster download  
- Backend integration with Google Apps Script  
- Automatic storage of posters in Google Drive  
- Feedback data logging to Google Sheets  

##  Tech Stack
- HTML5, CSS3, JavaScript  
- HTML Canvas API  
- Google Apps Script  
- Google Drive & Google Sheets  

## Usage
1. Open `feedback.html` using Live Server or Apps Script Web App  
2. Select event and image  
3. Enter participant details and feedback  
4. Preview and download the generated poster  

##  Purpose
Built for S2T 2026 to streamline feedback collection and generate professional event posters instantly.
